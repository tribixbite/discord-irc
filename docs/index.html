Get Started in Minutes

Deploy your Discord-IRC bridge with just a few simple steps. No complex configuration required.

1

Install the Bridge

Install via npm or clone from GitHub and build from source.

# Install via npm
npm install -g discord-irc-bridge

# Or clone from GitHub
git clone https://github.com/discord-irc/discord-irc.git
cd discord-irc && npm install && npm run build
2

Configure Your Bridge

Create a configuration file with your Discord token and IRC settings.

{
  "nickname": "DiscordBot",
  "server": "irc.libera.chat",
  "discordToken": "YOUR_DISCORD_BOT_TOKEN",
  "channelMapping": {
    "DISCORD_CHANNEL_ID": "#irc-channel"
  }
}
3

Start Bridging

Launch the bridge and start synchronizing messages between platforms.

# Start the bridge
npm start -- --config config.json

# Or use the global installation
discord-irc-bridge --config config.json
